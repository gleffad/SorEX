### How to install ###
* Compiling sources
```bash
cd src
javac -cp ./ util/*.java && javac -cp ./ blockchain/*.java && javac -cp ./ architecture/*.java
```
* Launching Client
```bash
java architecture/SorexClient
```
* Launching Server
```bash
java architecture/SorexClient
```
